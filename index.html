Sobre o projeto

O LIFTO é um sistema web desenvolvido para centralizar e simplificar o uso das ferramentas criadas para o gerenciamento de laboratórios de informática em ambientes educacionais.

O site do LIFTO será o ponto de partida para toda a operação do laboratório de informática. Ao centralizar as ferramentas, simplificamos o dia a dia e garantimos maior autonomia para as escolas.

Como usar

1. Acesso

Conecte-se à rede da escola e acesse o sistema através do navegador.

2. Navegação

Utilize o menu principal para acessar as diferentes funcionalidades.

3. Manutenção

Atualizações são automáticas. Para problemas técnicos, consulte a seção de contato.
